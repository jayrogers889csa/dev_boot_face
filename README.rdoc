== DevBootFace

Requirements:
Rails 4.0.2
Ruby 2.0.0p451

To run locally:
Clone repo to local machine
`bundle`

rake db:create
rake db:migrate
rails s
visit http://localhost:3000

Running the test suite:
In project root directory, run
ruby /spec/controllers/posts_controller_spec.rb
ruby /spec/controllers/users_controller_spec.rb
